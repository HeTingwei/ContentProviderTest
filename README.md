# ContentProviderTest
学习内容提供器的一个demo，包括访问其他应用的内容提供器和创建自己的内容提供器。<br>
有两个项目，reader为去操作另一个应用的应用，provider是有内容器被其他应用操作才的应用。<br>
得益于Android 7.0的新特性，下面是同时跑两个程序的效果，在reader中操作provider的数据库,在两个应用都可以查询数据库中的数据：<br>
